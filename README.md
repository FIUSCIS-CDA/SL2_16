#### MODULE NAME
       SL2_16

#### DEPENDENCIES
       SameBit

#### INPUTS
Name | Size(Bits) 
-----|------------
A   |     16     

#### OUTPUTS
Name | Size(Bits)
-----|------------
Y   |     18     

#### DESCRIPTION
       Shifts a 16-bit input A left by two slots, producing an 18-bit output Y
       = A * 4.
